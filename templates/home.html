Spygit

Embrace Standards

Spygit (pronounced 'spig-uht') is a the premier network to show off (or mourn) your Python project's standards.

Spygit removes the burden of manually checking your code against the pep8 standard and proving to your team/peers that you are compliant.

Why Standards?

  • Python publishes a community accepted code standard
  • Ensure all code looks the same
  • Help new developers concentrate on code, not style
  • The language is beautiful, your code should be too!
